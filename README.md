# Discord Music Bot

Vibe Bot Discord phát nhạc đơn giản, siêu mượt, nghe siêu phê.

# Cách chạy Bot:

Bước 1: Cài đặt thư viện cần thiết: ```pip install -r requirements.txt```

Bước 2: Dán đoạn plain text và thay token bot discord vào file .env: ```DISCORD_TOKEN=nhap_token_bot_cua_ban_vao_day```

Bước 3: Chạy bot: ```python bot.py```
